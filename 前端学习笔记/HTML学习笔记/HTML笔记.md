## QT html学习笔记

### 一、基本知识

#### 1.html页面的固定结构

```html
<!DOCTYPE html> //HTML5
<html lang="en"> //根标记开始
 
<head> //头标记开始
    <meta charset="UTF-8"> 
    <title>网页标题tle>
</head> //头标记结束

<body>//主体开始 
    网页主体内容..
</body>//主体结束
</html> //根标记结束
```

------

   

#### 2.语法规范

##### **1.html的注释**：     

**作用**：用于代码内容注解，写代码添加注释为了方便给自己和其他程序员理解功能含义

注释的快捷键：vs Code中：ctrl + /

**代码：**

```html
<!--这是html的注释--> 写在里面的内容不会显示在网页上只让开发人员看到
```

##### **2.html标签的结构**

标签结构： 

```html
<strong>文字变粗</strong>
<开始标签></结束标签>
```

标签由< 、>、/、英文组成，<><>有头有尾叫双标签，<>只有一个是单标签

**html标签与标签之间的关系可分为：**

**父子关系（嵌套关系）**

```html
<head>
	<title></title>
</head>
```

**兄弟关系（并列关系）**

```html
<head></head>
<body><body>
```

------



#### 3.排版标签

##### **1.标题标签**

**特性**：文字加粗，独占一行

```html
<h1></h1> 大
.....
....
..
<h6></h6> 小
```



##### **2.段落标签**

**作用：**用于文章文本排版，可以文字分段

**代码：**

```html
<p>我是一个段落标签</p>
```

**特性**：段落之间存在间隙，独占一行



##### **3.换行标签**

< br > 用于文字强制换行显示

```html
<br>
```

**特性**：单标签，让文字强制换行



##### **4.水平线标签**

< hr > 分割不同主体内容的水平线

```
<hr>
```

**特性**：单标签，在页面显示一条水平线

#### 5.媒体标签

##### **1.图片标签**

```html
< img src="" alt=""> 在网页中显示图片
alt 替换文本，在图标加载失败时 src 图片路径
```

特性：单标签，img标签需要进行属性的设置来展示对应的效果

**属性：**

src 图片

alt 替换文本

title提示文本：鼠标在图上悬停

width=“10px”和heingt=“15px” 高度个宽度，当两个属性只有一个时没有的属性会按比例缩放

**属性注意点：**

1.标签属性要写在开始标签内部

2.标签可以同时存在多个属性

3.属性之间空格隔开

4.属性之间没有顺序之分



##### 2.路径

需要使用各种文件时

**绝对路径**：指目录下的绝对位置，通常从盘符开始的路径

例如：c:\a\b\c\e

**相对路径**：从当前文件出发找目标文件的过程

**相对路径的分类**：

*· 同级目录* 在同一个文件夹< img  src="目标文件.gif"> < img src="./目标文件.gif">两种写法

·*下级目录* **目标文件**放在**和自己同级目录的文件夹** < img src="同级文件夹/目标文件.jpg">

·*上级目录*  目标文件在上级目录中  ../返回上一级  < img src="../目标文件.gif">







##### 3.音频标签

**代码：**

```html
<audio src="./music.mp3" contrls></audio>
```

**常见属性：**

| 属性名   | 功能                         |
| -------- | ---------------------------- |
| src      | 音频的路径                   |
| contrlos | 显示播放的控件               |
| autoplay | 自动播放（部分浏览器不支持） |
| loop     | 循环播放                     |

**注意点：**

音频标签目前支持三种格式：mp3、Wav、Ogg



##### 4.视频标签

**用于在页面中插入视频**

**代码：**

```html
<video src="./video.mp4" controls><\video>
```

**常见属性：**

| 属性名   | 功能                                      |
| -------- | ----------------------------------------- |
| src      | 音频的路径                                |
| contrlos | 显示播放的控件                            |
| autoplay | 自动播放（谷歌需要配合muted实现静音播放） |
| loop     | 循环播放                                  |

**注意点：**

视频标签目前支持三种格式：mp4、WebM、Ogg

#### 6.链接标签

< a > 超链接标签

**代码**：

```html
<a href="./目标网页.html">跳转</a>
<a href="#">空链接</a>
```

**特性：**

双标签，点击内容可以跳转到任何网页

**属性：target** **属性值：打开网页的形式**

| 取值   | 效果                                   |
| ------ | -------------------------------------- |
| _self  | 默认值，在当前窗口跳转（覆盖原来网页） |
| _blank | 在新窗口中跳转（保留原网页）           |
